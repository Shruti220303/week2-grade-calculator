# Student Grade Calculator

## Project Overview
This project is a Python program that calculates a student's grade based on their marks. The program asks the user to enter the student name and marks, then displays the grade along with an encouraging message.

## Objectives
- Practice Python decision making using if-elif-else
- Use loops for input validation
- Create reusable functions
- Display formatted output

## Setup Instructions
1. Install Python
2. Open the project folder
3. Run the program using:

python grade_calculator.py

## Grading Logic
A: 90-100  
B: 80-89  
C: 70-79  
D: 60-69  
F: 0-59

## Technical Requirements Implemented
- Used input() to get student name and marks
- Used variables to store data
- Used if-elif-else for grading logic
- Used while loop to validate marks (0–100)
- Created a function to calculate grade

## What I Learned
In this project I learned how to use conditional statements, loops, and functions in Python. I also learned how to validate user input and display structured program output.
